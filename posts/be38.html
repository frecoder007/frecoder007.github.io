Jupyter Notebook安装教程

2023-11-03参考资料：某乎1、某乎2

本文是自己留存，待换电脑时重新安装软件时用，如能帮到你，万分荣幸

Anaconda安装

首先从这个页面上安装anaconda，这里提供的是miniconda版本。

其次，常规来说，安装了Anaconda发行版时已经自动为你安装了Jupyter Notebook的，但如果没有自动安装，那么就在终端（Linux或macOS的“终端”，Windows的“Anaconda Prompt”，以下均简称“终端”）中输入以下命令安装：conda install jupyter notebook

安装numpy, pil库

在网页版jupyter notebook点击new，选择Terminal，输入pip install numpy

VMware虚拟机配置踩坑

2023-09-08今天在折腾虚拟机的时候尝试使用桥接模式将VMware Ubuntu和宿主机作为两台设备相互通信，但是在校园网环境下执行的时候，按照教程设置好了IP地址、网关、子网掩码后，发现可以ping通主机但无法ping通百度。查看了多个博客后，记录一个比较符合我情况的经验帖：

有关校园网的问题

我在学校宿主机的ip是10.4.18.20而网关是10.4.16.1子网掩码是255.255.252.0

我把虚拟机的ip设置成10.4.18.n，会出现可以ping的通8.8.8.8，ping外网的时候网址的ip也可以解析出来，但就是不出现包的

我的解决方式是把10.4.18.n改成了10.4.16.n才能ping通

但是问题又出现了，这个n也有要求比如10.4.16.22就可以ping外网，而10.4.16.20就不行和10.4.18.n情况一样。。。

我猜是虚拟机发出的包少了什么配置数据，所以被校园网的防火墙给劫了，导致返回不了数据，因为我用虚拟机ping宿主机，当宿主机的防火墙没有关的时候也是这样，一直没有数据返回。

于是我就让宿主机脸上wifi结果一切畅通，丝滑流畅。。。校园网真的是坑。。。

同样的，我改为个人热点连接后也成功ping了外网。证明校园网是会封锁某些ip数据的。至于更换ip没来得及尝试。顺带一提traceroute可以检测网络问题，奈何快断电了我也没来得及尝试。

同时有时候虚拟机和宿主机ping不通，检查自己的网络是公有网还是专用网络，此时把对应的防火墙关闭即可。
